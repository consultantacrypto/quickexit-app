# BMK Tier Calibration — Audit si plan

## 1. Context
BMK este token legacy Bitmarket. Quick Exit poate folosi BMK ca layer optional Legacy/VIP, fara expunere publica in aceasta etapa.

## 2. Ce exista acum
- BMK Lab privat la `/hq-admin/bmk-lab`.
- Connect wallet.
- Detectare BSC.
- Citire balanta BMK.
- Tier estimativ experimental.
- Fara swap.
- Fara plati BMK.
- Fara homepage exposure.

## 3. Tieruri curente experimentale
- `0`: Fara tier BMK
- `>0`: Legacy Watcher
- `>=1.000`: Legacy Member
- `>=10.000`: VIP
- `>=100.000`: Founder Circle

## 4. De ce trebuie calibrate
- Pragurile actuale sunt arbitrare.
- Este necesara analiza distributiei reale a holderilor.
- Trebuie excluse walleturi treasury / operationale / exchange cunoscute.
- Trebuie evitate interpretari de tip promisiune financiara.

## 5. Ce date trebuie colectate
- Lista holderilor.
- Balante holderi.
- Percentiles: P50 / P75 / P90 / P95 / P99.
- Top wallets.
- Walleturi treasury/exchange/operationale.
- Holderi activi vs dust holders.

## 6. Beneficii posibile non-financiare
- Acces early la functii noi.
- Badge intern.
- Prioritate suport.
- Discount utilitar controlat.
- Acces la features beta.
- Comunitate legacy.
