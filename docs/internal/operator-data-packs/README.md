# Operator Data Packs

Acest folder este rezervat pentru **exemple fictive** sau **exporturi manuale curate** destinate **Quick Exit Operator** (analiză internă read-only).

## Reguli

- **Nu** se comit date reale de utilizatori.
- **Nu** se comit PII (emailuri complete, telefoane complete, mesaje integrale, documente KYC).
- **Nu** se comit exporturi live brute din producție (dump-uri DB, export Stripe nemascat).
- Se pot comite **doar** exemple **fictive**, **șabloane** sau **structuri goale** (ex. subfoldere `YYYY-MM-DD/` populate manual după review).

## Format

Structura recomandată a unui pachet este documentată în:

`docs/internal/operator-manual-export-format.md`

## Exemplu de cale

`docs/internal/operator-data-packs/2026-05-12/` — conținut CSV/Markdown conform specificației, după validare.
